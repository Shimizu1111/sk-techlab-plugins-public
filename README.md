# sk-techlab-plugins-public

SK Tech Lab が公開する Claude Code プラグイン集です。

## インストール方法

```bash
# 1. マーケットプレイスを追加（初回のみ）
claude plugins marketplace add katsu/sk-techlab-plugins-public

# 2. 使いたいプラグインをインストール
claude plugins install <plugin-name>
```
